# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-09
### Added
- **Core Architecture:** Set up simple Python/Pandas backend and custom Streamlit frontend.
- **Design System:** Implemented full dual-theme CSS framework (variables, typography, glassmorphism cards).
- **Landing Page:** Premium, Apple/Stripe-inspired hero section.
- **Dashboard:** Real-time KPI cards fetching data via `yfinance`.
- **Analysis Page:** Interactive Plotly charts for stock history and financial statements.
- **Comparison Page:** Multi-select company comparison with normalized radar charts.
- **Buffett Score:** Algorithmic valuation feature.
- **Portfolio Simulator:** Interactive sliders and weighted metric calculations.
- **Documentation:** Added professional `README.md`, `VIVA_README.md`, and contribution guidelines.
